🧠 Smarter Automation (BEST PRACTICE)

Instead of blindly running cron:

👉 You can:

Find matches where:
matchDate < now
status = "upcoming"
Score them automatically
@Cron('*/10 * * * *') // every 10 mins
async autoScoreMatches() {
  const matches = await this.prisma.match.findMany({
    where: {
      matchDate: { lt: new Date() },
      status: 'upcoming',
    },
  });

  for (const match of matches) {
    await this.predictionService.scoreMatch(match.id);
  }

  await this.predictionService.updateUserPoints();
}
