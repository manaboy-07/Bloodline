# 🔥 1. CreatePrediction Mutation (Core Logic)

## 🧠 What we are building

👉 “Logged-in user predicts a match score”

* User → comes from JWT ✅

## ✅ GraphQL DTO (Input Type)

Create an input:

```ts
import { InputType, Field, Int } from '@nestjs/graphql';

@InputType()
export class CreatePredictionInput {
  @Field(() => Int)
  matchId: number;

  @Field(() => Int)
  homeScore: number;

  @Field(() => Int)
  awayScore: number;
}
```

---

## ✅ Mutation Resolver

```ts
import { Resolver, Mutation, Args, Context } from '@nestjs/graphql';
import { UseGuards } from '@nestjs/common';
import { PrismaService } from 'src/prisma/prisma.service';
import { PredictionType } from '../models/predictionmodel';
import { CreatePredictionInput } from '../dto/create-prediction.input';
import { GqlAuthGuard } from 'src/auth/gql-auth.guard';

@Resolver()
export class PredictionResolver {
  constructor(private prisma: PrismaService) {}

  @UseGuards(GqlAuthGuard)
  @Mutation(() => PredictionType)
  async createPrediction(
    @Args('input') input: CreatePredictionInput,
    @Context() context,
  ) {
    const userId = context.req.user.id;

    return this.prisma.prediction.create({
      data: {
        homeScore: input.homeScore,
        awayScore: input.awayScore,

        user: {
          connect: { id: userId },
        },

        match: {
          connect: { id: input.matchId },
        },
      },
      include: {
        match: true,
      },
    });
  }
}
```

# 🧠 What Just Happened (VERY IMPORTANT)

### This part:

```ts
user: {
  connect: { id: userId }
}
```

👉 Means:

> “Attach this prediction to the logged-in user”

---

### This part:

```ts
match: {
  connect: { id: input.matchId }
}
```

👉 Means:

> “This prediction is for this match”

### ✅ GraphQL Context Setup (IMPORTANT)

In your `GraphQLModule`:

```ts
GraphQLModule.forRoot({
  autoSchemaFile: true,
  context: ({ req }) => ({ req }),
});
```

---

### ✅ Your Guard (already showed but final version)

```ts
import { ExecutionContext, Injectable } from '@nestjs/common';
import { AuthGuard } from '@nestjs/passport';
import { GqlExecutionContext } from '@nestjs/graphql';

@Injectable()
export class GqlAuthGuard extends AuthGuard('jwt') {
  getRequest(context: ExecutionContext) {
    const ctx = GqlExecutionContext.create(context);
    return ctx.getContext().req;
  }
}
```
